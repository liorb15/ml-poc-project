# Assignment 1

## Mon projet
Prédiction du churn client pour un opérateur télécom.

## Le business case
L’objectif est d’identifier en amont les clients ayant un risque élevé de résiliation afin de permettre à l’entreprise de lancer des actions de rétention ciblées.

Un bon modèle peut aider à :
- réduire la perte de chiffre d’affaires ;
- prioriser les actions commerciales ;
- mieux comprendre les facteurs qui expliquent le départ des clients.

## Les sources de données
Je prévois d’utiliser un dataset public de churn télécom, par exemple le jeu de données "Telco Customer Churn", contenant des informations sur :
- le profil client ;
- le type d’abonnement ;
- l’ancienneté ;
- les services souscrits ;
- la facturation ;
- la variable cible de churn.

## Première idée de workflow
1. Nettoyer et préparer les données.
2. Séparer train / test.
3. Entraîner plusieurs modèles de classification.
4. Comparer leurs performances avec des métriques adaptées.
5. Présenter les résultats dans l’application Streamlit du template.
